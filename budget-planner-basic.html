04

Compatibility

  • File format: OOXML .xlsx.
  • Specific Excel versions and Google Sheets are not yet certified.
  • The current generator loads SheetJS 0.18.5 from cdnjs, so internet access or an already cached library is required during generation.

Privacy

  • This preview page does not collect financial data.
  • The current product build creates the blank workbook in the browser.
  • No account or form is provided on this preview.

Known limitations

  • Not publicly purchasable or downloadable yet.
  • No ExcelJS, project import/export or localStorage.
  • Workbook validation checks structure and formula families; it does not recalculate formulas.
05

Frequently asked questions

Is this a real Excel file?

Yes. The validated output is a genuine OOXML .xlsx workbook.

Can I use it fully offline?

We do not claim full offline support. Generation currently loads SheetJS 0.18.5 from cdnjs and therefore needs internet access or an already cached copy of that library.

Can I buy or download it now?

No public purchase or download route has been verified. Budget Planner Basic is shown as coming soon.

Which Excel versions are supported?

The generated file is a genuine .xlsx workbook, but specific Excel-version compatibility has not yet been certified.

Can I open it in Google Sheets?

Google Sheets compatibility has not yet been certified, so this preview does not make that claim.

What does the workbook contain?

Dashboard, Income, Expenses and Categories sheets, with SUM, SUMIF and SUMIFS formulas written into the workbook.
